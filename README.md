## About this GitHub

This GitHub is where I share notebooks I build in my free time. Nothing too crazy — or maybe sometimes there is something pretty good hiding in there.

Most of my work is related to **science**, **data analysis**, **visualization**, and sometimes **machine learning**. I try to keep every notebook clean, readable, and nicely commented so the reasoning is easy to follow.

## What you will find here

- Exploratory data analysis
- Scientific experiments and simulations
- Data visualization projects
- Machine learning experiments
- Well-commented notebooks made for learning and sharing

## My journey

I am a curious learner who enjoys using data to understand how things work. I like turning questions into small experiments, exploring datasets, testing ideas, and explaining the process clearly through notebooks.

This space is not meant to be perfect. It is a place to learn, improve, and keep track of projects that connect science, code, and curiosity.

## How I work

I try to make each notebook:

- Easy to read
- Clearly structured
- Properly commented
- Reproducible when possible
- Focused on the idea behind the analysis, not only the final result

## Thanks for reading

Thanks for stopping by. Feel free to explore the notebooks, learn from them, or get inspired for your own projects.
